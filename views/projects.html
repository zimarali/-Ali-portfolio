discGo is a application created for quick information on a music artist. It was developed using HTML, CSS, JavaScript, AngularJS, Bootstrap, and the Last.fm api. The page contains a search box used to retrieve info about a Music Artist once a name is entered into the search box. Upon submitting the form the information submitted is transfed to the information page. Once this information is transfered, the page retrieves information based on the entered artist name using the Last.fm api. To use the page click the demo button and check it out.

Theater Arck is a demo seat reservation application. It was developed using HTML, CSS, JavaScript, and Bootstrap. Seats a are aligned on the page to imitate a theater layout. Once a user clicks on a seat, a form will show up. Upon submitted the form the reservation is confirmed. Multiple seats can be reserved under the same information. To use the page click the demo button and explore.
